# Vista de Contexto

Actores externos:
- Cliente (persona que compra)
- Vendedor (usuario del sistema)
- Sistema de facturación externo (opcional)
- Base de datos persistente (local - SQLite)

Interacciones:
- Cliente: solicita registro, consulta productos, realiza compras.
- Vendedor: gestiona clientes, productos y registra ventas.
- Sistema de facturación: posible integración para emisión de facturas electrónicas.
